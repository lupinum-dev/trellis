# Architecture Decision Records

This directory contains accepted Trellis architecture decisions.

ADRs capture durable decisions that still shape the framework. They do not preserve every historical proposal, checklist, or planning artifact. Superseded RFCs and work plans should be deleted after their accepted decisions are represented here or in root foundation docs.
